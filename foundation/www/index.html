What is ERPNext?

ERPNext is a modern, fully featured open source Enterprise Resource Planning (ERP) system that is designed for small and medium sized businesses.

  • ERPNext includes modules to help you manage Financial Accounting, Sales and CRM, Billing, Manufacturing, Inventory, HR and Payroll, Projects, E-Commerce and much more.
  • ERPNext is designed to be web-first and is built for entreprenuers who want modern, clean design and great usablity. It is deeply integrated with email and other services and is designed to be deployed and managed on any remote server.
  • ERPNext is 100% free and open source (there are no paid modules) and is available under the GNU General Public License.

What is the ERPNext Foundation?

ERPNext Foundation is a not-for-profit company registered with the objective of building and enhancing ERPNext as an open source project. The ERPNext Foundation is funded by annual memberships of individuals and companies who use ERPNext or support it.

How do I Get Started?

If you are looking to self host ERPNext, you should start with a remote Virtual Machine (VM) and install ERPNext using the Easy Install Script.

Alternatively you can download the Virtual Machine and run it on your desktop

You can also sign up for a free one user account on ERPNext hosting services like erpnext.com
